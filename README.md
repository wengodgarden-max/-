# 🔮 决策炼金师 - 把你的判断力变成付费AI工具

一个独立的网页应用，帮助用户将专业判断能力提炼成可使用的AI工具。

## ✨ 功能特点

- 🎮 **游戏化流程**：通过4个关卡的引导式对话，轻松完成决策提炼
- 💎 **选题宝石**：锁定最擅长、最有价值的决策场景
- 🔍 **决策拆解**：将隐性思考过程显性化
- 📐 **结构化规则**：将经验转化为清晰的"如果...那么..."规则
- 🤖 **AI提示词生成**：一键生成可直接使用的AI提示词
- 🔗 **API支持**：提供标准API接口，支持集成到其他系统

## 🚀 快速开始

### 1. 启动服务

```bash
python src/main.py
```

服务默认运行在 `http://localhost:5000`

### 2. 访问网页应用

打开浏览器访问：
- 主页：`http://localhost:5000/`
- 决策炼金师页面：`http://localhost:5000/alchemist`

### 3. 开始游戏化流程

点击"快速模式"或"附件模式"开始您的决策提炼之旅！

## 📖 使用指南

### 关卡1：锁定宝藏选题

回答三个关键问题：
1. 您最擅长解决哪类问题？
2. 谁最需要这个能力？
3. 如果他们搞错了，会有什么后果？

### 关卡2：拆解您的决策

深入分析您的思考过程：
1. 第一步判断什么？
2. 关键区分标准是什么？
3. 决策转折信号有哪些？
4. 新手常见误区是什么？

### 关卡3：结构化为决策流

填写规则模板：
```
如果 [条件A] 成立
那么 属于类别 [类别名称]
建议动作 [具体行动]
风险点 [风险描述]
```

### 关卡4：封装成AI工具

系统自动生成：
- 完整的AI提示词
- 决策规则集
- 可直接复制使用的配置

## 🔌 API 接口

### 对话接口

```bash
POST /alchemist/chat
Content-Type: application/json

{
  "message": "我想创建一个决策工具...",
  "session_id": "user_123"
}
```

响应：
```json
{
  "status": "success",
  "message": "AI助手的回复内容",
  "session_id": "user_123",
  "run_id": "run_abc123"
}
```

### 标准运行接口

```bash
POST /run
Content-Type: application/json

{
  "messages": [
    {"type": "human", "content": "你好"}
  ]
}
```

### 流式运行接口

```bash
POST /stream_run
Content-Type: application/json

{
  "messages": [
    {"type": "human", "content": "请帮我创建一个决策工具"}
  ]
}
```
