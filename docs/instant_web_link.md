# 🌐 立即获得公开网页链接

## 最简单方案：ngrok（1分钟搞定）

### 步骤：

#### 1. 下载 ngrok（30秒）
打开浏览器，访问：
```
https://ngrok.com/download
```

选择你的系统：
- Windows: 下载 .zip 文件
- Mac: 下载 .dmg 或用 `brew install ngrok`
- Linux: 下载 .tar.gz

#### 2. 解压并运行（30秒）

**Windows:**
```
解压 ngrok.zip
双击 ngrok.exe
输入: ngrok http 5000
```

**Mac:**
```bash
brew install ngrok
ngrok http 5000
```

**Linux:**
```bash
unzip ngrok.zip
./ngrok http 5000
```

#### 3. 获得永久公开链接

你会看到类似这样的输出：
```
Forwarding  https://abc-123-xyz.ngrok-free.app -> http://localhost:5000
```

**你的公开链接：**
- 主页: https://abc-123-xyz.ngrok-free.app/
- 炼金师: https://abc-123-xyz.ngrok-free.app/alchemist
