## 第一步：创建 GitHub 仓库

### 操作步骤（1分钟）：

1. 打开浏览器，访问：https://github.com/new

2. 填写仓库信息：
   - **Repository name**: alchemist
   - **Description**: 炼金师助手 - 把你的判断力变成付费AI工具
   - **选择 Public**（公开）
   - **❌ 不要勾选**：
     - Add a README file
     - Add .gitignore
     - Choose a license

3. 点击绿色按钮 **"Create repository"**

---

## 第二步：推送代码

创建完成后，执行以下命令：

```bash
cd /workspace/projects

# 添加远程仓库
git remote add origin https://github.com/wengodgarden-max/alchemist.git

# 推送代码
git push -u origin main
```

---

## 第三步：Railway 部署

推送完成后：

1. 打开：https://railway.app
2. 点击右上角 **"Login"**
3. 选择 **"Login with GitHub"**
4. 点击 **"New Project"**
5. 选择 **"Deploy from GitHub repo"**
6. 选择 `alchemist` 仓库
7. 等待部署完成（2-3分钟）
8. 在 Settings → Domains 生成永久链接

---

## 最终链接格式

部署完成后获得：
```
https://alchemist-production-xxxx.up.railway.app/alchemist
```
