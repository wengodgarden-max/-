## 第三步：Railway 部署（2分钟）

### 3.1 登录 Railway
1. 打开：https://railway.app
2. 点击右上角 **"Login"**
3. 选择 **"Login with GitHub"**
4. 点击 "Authorize Railway"

### 3.2 创建项目
1. 点击 **"New Project"**
2. 选择 **"Deploy from GitHub repo"**
3. 选择你的 `alchemist` 仓库
4. Railway 自动开始部署（2-3分钟）

### 3.3 获得永久链接
1. 部署完成后，点击 **"Settings"**
2. 滚动到 **"Domains"**
3. 点击 **"Generate Domain"**
4. 获得永久链接：

```
https://alchemist-production-xxxx.up.railway.app
```

**你的访问地址：**
- 主页：https://你的链接/
- 炼金师：https://你的链接/alchemist
